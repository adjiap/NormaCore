# Security Policy

## Reporting a Vulnerability

Please do **not** report security vulnerabilities through public GitHub issues.

Send an email to [adjiap](20769489+adjiap@users.noreply.github.com). You can
expect an acknowledgement within 5 business days.

## Scope

This policy applies to the NormaCore repository only. Vulnerabilities in
upstream dependencies (Qdrant, Ollama, Docling) should be reported to their
respective maintainers.
